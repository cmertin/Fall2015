Linear Classifiers

←Previous lecture  |  Home  |  Next lecture →

This lecture is the first of several lectures dealing with linear classifiers. The linear model is the most popular classifier and over the course of many lectures in the class, we will see different learning algorithms for this hypothesis class.

This lecture defines a linear classifier and points out that it is an expressive hypothesis class. It ends with the first of many algorithms for learning linear classifiers – the least mean square method.
