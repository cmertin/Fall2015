Online Learning and the Perceptron Algorithm

←Previous lecture  |  Home  |  Next lecture →

This lecture discusses online learning and how we might quantify performance of an online learning algorithm using the mistake bound approach. Then, we look at the very widely used Perceptron algorithm and its mistake bound theorem.
